Android App: (χρησιμοποιήθηκε Android Studio)
-----------
 
Έχει ένα κουμπί για χειροκίνητη ενεργοποίηση και απενεργοποίηση ήχου και φλας, τα οποία ανόιγουν και κλείνουν μαζί.
Για να λειτουργήσει o MQTT Client με emulator θέλει: tcp://10.0.2.2:1883.
Για να λειτουργήσει στο κινητό θέλει την ip του τοπικού δικτύου (υπολογιστή).
Πατώντας το home button αν το κουμπί ήταν On εξακολουθεί να λειτουργεί ο φακός και ο ήχος.

Java App: (χρησιμοποιήθηκε IntelliJ)
---------

Ανοίγoντας πρώτα το android app και μετά το java app δίνουμε τις εξής εντολές μέσω terminal στο java app:
turn On (για άνοιγμα του φακού και του ήχου)
turn Off (για κλείσιμο του φακού και του ήχου)
finish (για τερματισμό της java app)
